# Magic Bytes

**Category:** Forensics
**Difficulty:** Beginner

## Description

A file has been renamed with the wrong extension. Identify its true type using magic bytes and open it correctly.

**Hint:** The true file is an image. Once you give it the correct `.png` extension and view it, the flag is printed right on the picture.

## Objective

Find the flag.

## Challenge Files

- `challenge-files/forensics/magic-bytes/not_a_png.xyz`

Download these files to begin the challenge.

## Flag Format

`CTF{...}` — submit this flag after solving the challenge.
