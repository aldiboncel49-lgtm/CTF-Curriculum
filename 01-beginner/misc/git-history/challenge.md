# Git History

**Category:** Misc
**Difficulty:** Beginner

## Description

A git repository has a flag committed in the past but later removed. Recover it from the commit history.

## Objective

Find the flag.

## Challenge Files

- `challenge-files/misc/git-history/` — Entire `.git` directory with full commit history

Download the entire directory and use `git log` / `git checkout` to recover the deleted flag.

## Flag Format

`CTF{...}` — submit this flag after solving the challenge.
