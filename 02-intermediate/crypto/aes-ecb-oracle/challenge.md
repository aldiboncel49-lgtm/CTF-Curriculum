# AES-ECB Oracle

**Category:** Crypto
**Difficulty:** Intermediate

## Description

An encryption oracle appends the flag to your input then encrypts with AES-ECB. Use a chosen-plaintext attack to recover the flag byte by byte.

## Objective

Find the flag.

## Challenge Files

- `challenge-files/crypto/aes-ecb-oracle/oracle.py`

Download these files to begin the challenge.

## Flag Format

`CTF{...}` — submit this flag after solving the challenge.
