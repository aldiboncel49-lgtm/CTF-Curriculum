# FactorDB Attack

**Category:** Crypto
**Difficulty:** Intermediate

## Description

An RSA public key has a modulus that can be factored using FactorDB. Decrypt the ciphertext to recover the flag.

## Objective

Find the flag.

## Challenge Files

- `challenge-files/crypto/factordb-attack/challenge.json` — RSA parameters (n, e, ciphertext)

Download these files to begin the challenge.

## Flag Format

`CTF{...}` — submit this flag after solving the challenge.
